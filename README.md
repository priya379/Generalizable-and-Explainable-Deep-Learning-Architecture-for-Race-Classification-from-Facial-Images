# Generalizable-and-Explainable-Deep-Learning-Architecture-for-Race-Classification-from-Facial-Images
This paper proposed a robust deep learning approach for race classification using a MobileNetV3-Large backbone with custom layers. The two-stage training strategy yielded better intra-dataset and cross-dataset accuracies using the FairFace and UTKFace datasets. It demonstrating strong generalization. Grad-CAM visualizations explain model decisions.This method outperforms existing state-of-the-art methods.It demonstrates robustness for real-world applications such as surveillance, security, and biometric verification.
